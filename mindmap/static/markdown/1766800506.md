# DifyApp 系统文档

## 项目概述

### 项目特点
- 企业级架构：Spring Boot 3.x + Vue 3
- 多租户支持：细粒度权限控制
- RAG 技术：LangChain4j 实现
- 流式响应：支持实时交互
- 多模态支持：文本/图片输入，集成 OCR
- 可扩展性：模块化设计
- 跨平台：Web + Tauri 桌面应用
- 丰富集成：MCP 协议、多种向量数据库、LLM

### 核心功能
- 用户认证与授权
- 智能对话：Chat Flow & Workflow
- 知识库管理：多格式解析与向量化
- RAG 问答：语义检索与引用追溯
- AI 应用管理：创建、配置、统计
- Text2SQL：自然语言转 SQL
- AI 绘图：Mermaid 图表生成
- 系统管理：配置、模型、日志
- MCP 协议支持：搜索、位置、时间服务
- OCR 服务：图片文字识别

## 系统架构

### 前端层
- Vue 3，Port: 3000

### 后端层
- Spring Boot，Port: 9090

### 数据存储层
- 关系型数据库：PostgreSQL/MySQL
- Redis：缓存
- 向量数据库：Chroma/FAISS/Milvus/Qdrant/Weaviate/PgVector/Elasticsearch
- MinIO：对象存储

### 外部服务
- Dify API：LLM 服务

## 技术栈

### 后端技术栈
- 核心框架：Spring Boot 3.5.8
- 编程语言：Java 17

## 快速开始

### 环境要求
- **后端**：JDK 17+，Maven 3.6+，数据库，Redis（可选），MinIO，向量数据库
- **前端**：Node.js 16+，npm/yarn，Git
- **Tauri**：Rust 1.70+（可选）

### 1. 克隆项目
```bash
git clone https://github.com/Yarao-Liu/DifyApp.git
cd DifyApp
```

### 2. 后端启动
- 配置 `application.yml`：数据库、Redis、MinIO、向量数据库
- 执行初始化 SQL 脚本
- 构建并运行：`mvn clean install`，`mvn spring-boot:run`

### 3. 前端启动
- 安装依赖：`npm install`
- 配置 API 地址：`frontend/src/config/api.js`
- 启动开发：`npm run dev`

### 4. 访问系统
- 前端：`http://localhost:3000`
- 后端 API：`http://localhost:9090`
- API 文档：`http://localhost:9090/swagger-ui.html`

## 系统数据流

### 交互流程
1. 用户访问 → 前端 → 认证服务 → 数据库验证 → JWT 返回
2. 创建知识库 → 上传文档 → 保存元数据 → 向量化存储
3. 发起对话 → 前端 → 对话服务 → 检索知识库 → 向量数据库 → LLM 调用 → 返回结果

## 功能模块关系

### 模块层级
- **用户层**：用户认证模块
- **应用层**：智能对话、知识库、AI 应用、系统管理
- **服务层**：Text2SQL、MCP 协议、OCR 服务

### 依赖关系
- 认证 → 所有模块
- 智能对话 → 知识库、MCP、OCR
- 知识库 → 系统管理
- 系统管理 → Text2SQL

## 功能模块

### 1. 用户认证模块
- **后端**：注册/登录/登出、JWT、密码安全、权限控制、可见性管理、管理员审核
- **前端**：登录页、密码修改、令牌自动管理

### 2. 智能对话模块
- **后端**：应用管理、对话接口、流式响应、上下文管理、MCP、OCR 集成
- **前端**：实时聊天、模式切换、Markdown 渲染、会话管理

### 3. 知识库模块
- **后端**：创建/编辑/删除、文档解析、分块、向量化、RAG 问答、版本管理
- **前端**：上传界面、处理状态、问答界面、统计展示

### 4. AI 应用管理模块
- **后端**：应用生命周期、配置管理、可见性、使用统计、模板管理
- **前端**：应用列表、配置界面、统计图表

### 5. 系统管理模块
- **后端**：系统配置、数据源/模型/向量库管理、Prompt 模板、Text2SQL、用户管理
- **前端**：配置界面、管理面板、用户审核、权限设置

### 6. 其他功能
- 主题切换：深色/浅色，VS Code 风格
- Markdown 渲染：代码高亮、公式、图表
- 帮助文档：内置引导
- 响应式布局：适配移动端
- 桌面应用：Tauri 支持
- API 缓存：智能缓存
- 错误处理：统一提示
- 国际化：预留接口

## 部署说明

### 后端部署
- 打包：`mvn clean package`
- 运行：`java -jar target/backend-0.0.1-SNAPSHOT.jar`
- 生产建议：HTTPS、JWT 密钥加固、日志配置、监控告警

### 前端部署
- 构建：`npm run build`
- 部署：将 `dist/` 部署至 Nginx/Apache
- Tauri 构建：`npm run tauri:build`

- OCR 服务：`easy_ocr/docker-compose
