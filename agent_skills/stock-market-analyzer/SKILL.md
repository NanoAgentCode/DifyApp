# 港股A股市场行情分析工具

## 核心能力

实时获取和分析港股A股市场行情数据，提供：
- 大盘指数（上证、深证、恒指）
- 个股涨跌情况
- 板块表现分析
- 市场情绪判断
- 涨跌原因解读

## 数据获取方式

**推荐方式**：使用浏览器访问新浪财经实时行情页面

```
上证指数：https://finance.sina.com.cn/realstock/company/sh000001/nc.shtml
深证成指：https://finance.sina.com.cn/realstock/company/sz399001/nc.shtml
恒生指数：https://finance.sina.com.cn/realstock/company/HSI/nc.shtml
```

### 浏览器调用流程

1. **启动浏览器**
```javascript
browser action=start
```

2. **打开行情页面**
```javascript
browser action=open url="https://finance.sina.com.cn/realstock/company/sh000001/nc.shtml"
```

3. **获取页面快照**
```javascript
browser action=snapshot targetId="{page_id}"
```

4. **解析行情数据**
从快照中提取：
- 当前点位
- 涨跌幅
- 成交量
- 成交额
- 板块表现
- 个股涨跌榜
